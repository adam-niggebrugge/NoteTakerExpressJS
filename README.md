## Technology

## Installation

```

npm install express
npm install uuid

```

For local deployment
```
node server.js
```

I used Heroku for a cloud server. The following commands do require an account to be started with Heroku.
```

heroku login
heroku auth:login
heroku create
git remote -v

```
